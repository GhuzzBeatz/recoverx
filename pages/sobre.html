RecoverX
Recuperacao profissional de arquivos para Windows - GHZ Plugin
O que e o RecoverX

O RecoverX e um software desktop para Windows que ajuda a recuperar arquivos deletados. Ele atende cenarios comuns, como exclusao acidental de fotos e documentos, e cenarios mais complexos, como disco formatado.

Funcionalidades
SR
Scan Rapido
Escaneia lixeira, pastas temporarias e cache para encontrar arquivos deletados recentemente em poucos segundos.
SP
Scan Profundo
Combina PhotoRec com Windows File Recovery quando disponivel. Isso aumenta a chance de encontrar arquivos em exclusao recente e tambem em varredura bruta apos formatacao.
RS
Resultados
Visualizacao com filtros por tipo e copia seletiva dos arquivos encontrados para outra pasta de seguranca.
LJ
Limpeza de Junk
Remove arquivos temporarios e cache para liberar espaco no sistema com seguranca.
