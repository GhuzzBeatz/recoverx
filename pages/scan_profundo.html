Scan Profundo

Recuperacao avancada usando PhotoRec e Windows File Recovery para buscar mais arquivos no disco.

Nenhum motor de scan profundo encontrado
Para usar esta area, instale ao menos um dos motores abaixo:

PhotoRec: coloque photorec_win.exe ou testdisk_win.exe na pasta tools\ do app.
Windows File Recovery: instale pela Microsoft Store no proprio Windows.

Quando os dois estiverem disponiveis, o RecoverX libera o modo Hibrido, que costuma achar mais arquivos.
Motor de recuperacao
O RecoverX pode usar um ou dois motores no scan profundo. Quando o modo Hibrido estiver disponivel, ele combina os achados do Windows File Recovery com o PhotoRec.
